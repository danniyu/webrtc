WebRTC Videochat Demo
=====================

![](<server/public/images/webrtc-logo.png>)


Technologies used: 
-------------------

1.  NodeJS
2.  WebRTC

Browser supported: 
-------------------

1.  Webkit based
2.  Gecko/Mozilla based
3.  Microsoft IE 10+

How to run:
-----------

1. Run `node app.js`
2. Go to your browser `http://localhost:2000`
